### 

Profissional focado em Engenharia de Software, com especialização no desenvolvimento de sistemas backend escaláveis, análise de dados e implementação de soluções de Inteligência Artificial. Experiência na concepção de arquiteturas de dados, modelagem de bancos de dados relacionais e não-relacionais, e criação de dashboards para inteligência de negócios (BI).

**Áreas de Atuação Principal:**

- **Desenvolvimento Backend** — Criação de APIs robustas e arquiteturas escaláveis com Django, Laravel e FastAPI.
- **Inteligência Artificial** — Implementação de modelos e soluções baseadas em IA para otimização de processos.
- **Engenharia de Dados** — Processamento de dados estruturados e não-estruturados, criação de dashboards interativos e relatórios analíticos.
- **Arquitetura de Banco de Dados** — Modelagem e otimização de bancos de dados para alta performance.
